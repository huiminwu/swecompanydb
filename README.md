# swecompanydb

To run locally,

Install packages in requirements.txt with ```pip install -r requirements.txt```

In swecompanydb/settings.py, 
- change SECRET_KEY to your secret key (generated by django)
- change DEBUG to TRUE (optional)
- If you're running into database locking issues, you can try to add your specific mysql user info in the DATABASES field
  - [this might help](https://www.digitalocean.com/community/tutorials/how-to-create-a-django-app-and-connect-it-to-a-database)
  
Make sure to migrate models!
